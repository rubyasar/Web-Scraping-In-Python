# Web-Scraping-in-Python

This repository makes use of BeautifulSoup, urllib and re to extract data from us analytics website in a very simple manner and save to different text files.
Functions/constructs used:
  -find and find_all() to search tags and text in source code.
  -open and write for saving results in files.
  -'lxml' is the parser used in BeautifulSoup Constructor
  -urlopen to open weblink
